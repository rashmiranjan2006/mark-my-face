Mark My Face - Surveillance Attendance System
Mark My Face is a modern attendance management system that uses face recognition technology for secure, contactless, and efficient student attendance tracking. Built with React, Vite, Tailwind CSS, and shadcn-ui, it provides a seamless user experience for both students and administrators.

Features
Face Recognition Attendance: Capture and verify student attendance using facial recognition.
Student Registration: Students can sign up and upload their photo for future recognition.
Admin Dashboard: Administrators can view, manage, and mark attendance records.
Responsive UI: Clean, accessible, and mobile-friendly interface.
Supabase Integration: Secure backend for authentication, user management, and data storage.

Tech Stack
React (JavaScript)
Vite
Tailwind CSS
shadcn-ui
Supabase
